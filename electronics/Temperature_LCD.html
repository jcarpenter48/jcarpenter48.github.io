XPS 435T/9000 RI0707 Overclocking

2023-04-12

I salvaged a GC-6052S segment display from an old alarm clock quite some time ago, and I finally decided to put it to use as a temperature readout display for my computer. The components of this project were threefold: first, write the embedded program on an arduino to talk to the display; second, write the tray application to talk to the arduino; third, design and print a bracket to hold the display.

Thankfully, I was able to find some resources online that helped simplify this project, one that documented displays like the GC-6502s, an example project in the same vein, and of course, not least, the OpenHardwareMonitor project, which allowed me to easily write a tray application to send serial data to the Arduino for stats monitoring.

I have since evolved this project more, and must add some more details of the project when I have time. -10/2023
